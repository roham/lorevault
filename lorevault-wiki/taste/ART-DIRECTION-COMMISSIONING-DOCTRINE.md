## 0. The Problem This Solves

The prior generation sessions produced 20 BS-1 base art cards with inconsistent parallel tagging in the manifest and ARCANA-adjacent visual elements (sacred-geometry mandala, alchemical geometry, accidental sigil patterns) bleeding into Common-tier and base-card prompts because the CEO once noted the ARCANA look was visually striking.

**The failure mode**: ARCANA becomes wallpaper. Every card looks vaguely mystical. The ARCANA parallel — the sacred-geometry mandala behind the figure's head — loses its power as a chase variant because collectors see it everywhere.

**The fix**: ARCANA is a PARALLEL, not a house style. Parallels are TIER-GATED (Rare+ only) and ART-TREATMENT-SPECIFIC (the visual signature lives ONLY in the parallel variant, never in base art). This doctrine enforces that distinction with a hard pre-commission gate.

## 1. Tier-Density Rubric

Visual complexity scales with rarity. Cheaper cards are plainer. This is not cost-cutting — it is design.

**The logic**: Common cards are the majority of the pack. They must be individually beautiful but compositionally minimal — a reader skipping through 12 Commons needs a clear subject, legible at thumbnail, no visual noise. Rare and above earn their additional complexity by being genuinely harder to look away from.

### Tier 1 — Common (15,000 mint)

**Density class: PLAIN**

- **Subject**: Single figure OR single object (absent-character trace). No more than 2 figures in frame.
- **Environment**: Immediate context only. Baker Street domestic interior; forest shadow; Victorian staircase; single atmospheric element. Not a sweeping landscape.
- **Atmosphere**: Universe-specific base tone (gaslight fog for Baker Street; bone-cold forest for Enchanted Kingdom; etc.) applied at low register — present but not dramatic.
- **Composition**: Classic illustrative framing. Subject fills the middle third. One strong negative-space element allowed.
- **Lighting**: One primary source, one ambient fill. No non-diegetic (Alvinized) light on Common. The world as it is.
- **Symbolic complexity**: One anchor object (the Persian slipper, the notebook, the telegram). One iconographic layer. Not two.
- **FORBIDDEN on Common base art**: Sacred-geometry mandala; alchemical sigils; golden occult filigree; cosmic aurora; nebulae; divine particulate; god-rays; synthwave palette; near-future cyber elements; any element that reads as "parallel signature."

### Tier 2 — Rare (2,500 mint)

**Density class: MODERATE**

- **Subject**: One strong figure or a dyad (protagonist + witness or protagonist + foil). Secondary figures allowed soft-focus at periphery.
- **Environment**: Charged atmospheric environment. Baker Street at full fog-depth; forest at dusk with implication of what's in the trees; Gothic staircase with shadow architecture. Environment as biography.
- **Atmosphere**: Universe tone at full expression. The moment feels distinctive.
- **Composition**: Elevated beyond Common. One distinctive compositional move: elevated perspective, dramatic depth, scale contrast, relational triangle. The card has a reason it earns the Rare slot.
- **Lighting**: One Alvinized source is PERMITTED on Rare base art — a warm gold column from an impossible direction, or a lamp placed where no lamp could be. One non-naturalistic touch only.
- **Symbolic complexity**: Two iconographic layers. Primary anchor + a secondary hidden detail that rewards a second look.
- **FORBIDDEN on Rare base art**: Sacred-geometry mandala; alchemical sigils; cosmic aurora; nebulae; divine particulate; synthwave palette; near-future cyber elements. The Alvinized permitted exception is ONE non-diegetic light source — it is NOT a mandala, it is NOT sigils.

### Tier 3 — Legendary (250 mint)

**Density class: RICH**

- **Subject**: The definitive figure in the definitive moment. Fans remember this by name. The figure is unmistakably itself — the silhouette-recognition test (Silhouette layer of Lampblack) must pass instantly.
- **Environment**: Full environmental storytelling. The room, the landscape, the sky, the objects — all are biographical. Every visible element was placed deliberately.
- **Atmosphere**: Universe-registered but elevated. Struzan orbital composition or a museum-quality visual argument.
- **Composition**: Multiple compositional elements working together. Orbital (main figure centered, supporting cast orbiting), or Relational Triangle (protagonist + foil + separating world), or Scale Contrast (tiny figure, mythic sky). The composition carries a narrative argument by itself.
- **Lighting**: Multiple legitimate Alvinized sources. The Legendary card earns non-diegetic light as part of its mythologizing register. The Lampblack of the figure can glow, if it never glows on their Common cards.
- **Symbolic complexity**: Three or more iconographic layers. A primary anchor, a secondary hidden detail, and a tertiary environmental reading that only rewards the collector who has read the flavor text.
- **FORBIDDEN on Legendary base art**: Sacred-geometry mandala; alchemical sigils; cosmic aurora; nebulae; divine particulate; synthwave palette; near-future cyber elements. Alvinized light is permitted; ARCANA visual language is not the same as Alvinized light.

### Tier 4 — Ultimate (25 mint)

**Density class: MAXIMALIST**

- **Subject**: THE defining moment of the Set. One per Set. Not a strong moment — THE moment. Holmes meeting Watson. The wolf at the grandmother's door. The first bite of the apple. Persephone's descent.
- **Environment**: Full narrative environment at maximum density. Struzan theatrical grandeur + Alex Ross photo-real mythic posture as the ceiling. Every square centimeter of the frame is load-bearing.
- **Atmosphere**: Museum-quality. Drew Struzan orbital composition at its fullest expression. The card commands space.
- **Composition**: The Ultimate card earns complexity as its defining virtue. No single dominant technique — all of them, in their correct hierarchy: figure hierarchy (primary → secondary → tertiary), environmental depth, scale contrast, Alvinized mythologizing light, implied off-frame, Lampblack recognition.
- **Lighting**: Full Alvinized treatment. The impossible light on the central gesture (the handshake, the first bite, the held moment) IS the Ultimate's signature.
- **Symbolic complexity**: All layers simultaneously legible at different timescales. First glance: the iconic figure. Second look: the secondary character and the environmental staging. Third look: the hidden detail that only a reader of the flavor text would find.
- **FORBIDDEN on Ultimate base art**: Sacred-geometry mandala; alchemical sigils; cosmic aurora; nebulae; divine particulate; synthwave palette; near-future cyber elements. The Ultimate is maximalist in NARRATIVE density, not PARALLEL-signature density. It earns its weight through compositional sophistication, not through mystical overlays.

## 2. Parallel Reservation Rules

**The core rule**: Parallel visual signatures are EXCLUSIVE to their designated parallel variant. They do NOT appear on base art, ever, for any reason, regardless of tier.

A parallel is a distinct product. The collector who pays for the ARCANA parallel is paying for the sacred-geometry mandala that the base card does NOT have. If the base card already has ARCANA visual elements, the parallel is not worth buying.

### Base Art (any tier) — PARALLEL = BASE

Visual treatment: ART-DIRECTION-V3.md V3 template, with the tier-density modifier from §1 above. Universe-specific visual DNA per §1 of ART-DIRECTION-V3.md (Paget linework for Baker Street, Tenniel for Wonderland, etc.).

FORBIDDEN across all base art: any visual element that belongs to a parallel.

The FORBIDDEN list for base art — these elements must not appear in any base card prompt:

```
FORBIDDEN PARALLEL SIGNATURES (base art):
- Sacred-geometry mandala behind character's head
- Alchemical sigils (geometric glyph-patterns, runic overlays)
- Golden occult filigree / golden geometric border treatments
- Cosmic aurora or aurora borealis as background element
- Nebulae, starfield, cosmic dust-cloud as background element
- Divine particulate (floating golden/silver particles of mystical origin)
- God-rays of non-diegetic mystical origin (note: a lamp can cast god-rays;
  an empty sky cannot)
- Synthwave palette (hot pink, electric cyan, neon violet)
- Near-future cyberpunk architecture, holographic UI elements, neon signage
```

### ARCANA Parallel (Rare+ only) — PARALLEL = ARCANA

**The sacred-geometry variant.** The collector hunting ARCANA is hunting the mandala.

Visual signature — ALL of the following must be present in the ARCANA prompt, layered over the base-tier composition:
- A sacred-geometry mandala glowing behind the character's head (the primary element — this IS the ARCANA)
- Alchemical sigils worked into the background — geometric, glowing, not decorative
- Golden occult filigree framing the composition (can merge with the card-border treatment)
- Cosmic aurora in the background (universe-toned: gaslight-gold aurora for Baker Street, emerald-forest aurora for Enchanted Kingdom, ether-blue for Gothic Horror, rose-dawn aurora for Greek Myth, dream-violet aurora for Wonderland)

Tier modifier for ARCANA:
- **ARCANA Rare**: Mandala present but modest in scale. Sigils legible but not dominant. The base composition still reads beneath the parallel treatment.
- **ARCANA Legendary**: Mandala fills the upper half of the frame. Sigils form a secondary pattern. The filigree begins to merge with environmental elements.
- **ARCANA Ultimate**: Full-field sacred geometry. The mandala IS the sky. The sigils are integrated into the environmental architecture. The figure stands inside the geometry as if the geometry is native to them.

Permitted on: ARCANA parallel cards with Rare / Legendary / Ultimate tier only.
FORBIDDEN on: Common cards (any parallel). Non-ARCANA parallels. Base art at any tier.

### AETHER Parallel (Rare+ only) — PARALLEL = AETHER

**The mythic-cosmic variant.** The figure as subject of cosmic scale.

Visual signature — ALL of the following present:
- Aurora-cosmic backdrop: nebulae, galactic drift, cosmic dust (universe-toned as above)
- Divine particulate: floating luminous motes (gold for Baker Street / Greek Myth; silver for Wonderland; crimson for Gothic Horror; emerald for Enchanted Kingdom)
- God-rays of non-diegetic cosmic origin — the figure lit from above/behind by a light source that cannot exist in the physical scene
- Scale expansion: the background pushes to cosmic scale; the figure is simultaneously intimate and mythic

AETHER is the "MYTHIC" register. The figure is a small thing against a vast cosmos.

Permitted on: AETHER parallel cards with Rare / Legendary / Ultimate tier only.
FORBIDDEN on: Common cards (any parallel). Non-AETHER parallels. Base art at any tier.

### WITNESS Parallel (Rare+ only) — PARALLEL = WITNESS

**The photoreal documentary variant.** The moment as if filmed.

Visual signature — ALL of the following present:
- Photoreal, documentary-cinema realism — Deakins / Lubezki register
- Period-accurate material texture (fabric, leather, skin, stone — all at photographic resolution)
- Cinema vérité composition: handheld-camera-implied frame, naturalistic blocking, practical light only
- No mythologizing: no Alvinized light, no impossible angles, no scale contrast. The world as a camera would see it.

WITNESS is the "this actually happened" register. Where base art romanticizes, WITNESS makes it feel like evidence.

Permitted on: WITNESS parallel cards with Rare / Legendary / Ultimate tier only.
FORBIDDEN on: Common cards (any parallel). Non-WITNESS parallels. Base art at any tier.

### NEON Parallel (Rare+ only) — PARALLEL = NEON

**The cyber-modern reimagining variant.** The figure in a 2040s context.

Visual signature — ALL of the following present:
- Character relocated to a near-future urban-gothic setting
- Synthwave palette (hot pink, electric cyan, neon violet, electric green accents)
- Modern/cyber architecture: glass towers, neon signage, holographic interfaces
- Character's Lampblack survives: the figure is unmistakably the same character, Spine intact, despite the setting change (the deerstalker becomes a AR visor; the pipe becomes an atomizer; but the posture, the facial architecture, the elemental remain)

NEON is the "same figure, different shell" register. It maps to the CYBER Shell in the Lattice taxonomy.

Permitted on: NEON parallel cards with Rare / Legendary / Ultimate tier only.
FORBIDDEN on: Common cards (any parallel). Non-NEON parallels. Base art at any tier.

### ONE-OFF Parallel (Legendary / Ultimate only) — PARALLEL = ONE-OFF

**Bespoke, single-instance commissioned variant.** No template — manually crafted through high-iteration generation. Not in packs. Separate curated offering.

Daemon does not auto-generate ONE-OFFs. ONE-OFFs require Council Art Director review and manual iteration outside the Track A pipeline.

## 4. The ARCANA Overprevalence Audit (BS-1 Base Art)

A review of the 20 base art prompts against the FORBIDDEN list above reveals three cases requiring attention before parallel generation begins:

**bs1-c05 "The Chemistry Bench"** (Common): Prompt reads: *"the shadow's silhouette is subtly wrong: where Holmes's sleeve is pushed back, the shadow shows a cuff, buttoned."* — This is acceptable as a subtle hidden detail (Forbidden register check: PASS). No sacred geometry. No sigils. PASS.

**bs1-r01 "V.R. in the Plaster"** (Rare): Prompt reads: *"a mandala of impact craters radiating from the letters — not literal, but the wallpaper pattern around the holes forms an accidental geometry."* — Caution. The word "mandala" and the accidental geometry are borderline. If the generated image shows a literal geometric mandala pattern, this conflicts with doctrine. If the image shows a naturalistic scatter of bullet damage, it is fine. The ARCANA parallel of this card, when commissioned, must visibly UPGRADE beyond the base — if base art has mandala-adjacent elements, the ARCANA parallel loses distinction. **Recommendation**: When commissioning bs1-r01 ARCANA parallel, verify that the base art does not already show explicit sacred geometry. If it does, request re-generation of the base card with a cleaned prompt. Accept as-is pending visual review.

**bs1-r04 "The Case Map"** (Rare): Prompt reads: *"a ARCANA geometry of deduction laid bare."* — The word "ARCANA geometry" appears in the style line as flavor. The actual visual requested is a top-down overhead view of papers radiating from Holmes. This is an overhead compositional choice, not a sacred-geometry parallel treatment. PASS — but the prompt should not use the word "ARCANA" in future base art generation. Logged for future prompt template cleanup.

**Conclusion**: No re-generation required for the 20 existing base art cards. The ARCANA problem has been prevented from reaching the generated output level, though it was present in the prompt-writing intent. This doctrine prevents recurrence.

## 5. Pre-Commission Gate — Track A Hard Check

Before any image generation call (`genReplicate()` or equivalent), the daemon MUST verify the following checklist. Any FAIL blocks generation.

```
PRE-COMMISSION CHECKLIST

1. CARD IDENTITY
   [ ] Moment ID confirmed (e.g., bs1-r01)
   [ ] Set confirmed (e.g., bs-1-221b)
   [ ] Tier confirmed: Common / Rare / Legendary / Ultimate
   [ ] Parallel confirmed: BASE / ARCANA / AETHER / WITNESS / NEON / ONE-OFF

2. TIER-PARALLEL COMPATIBILITY
   [ ] If Parallel = BASE: any tier → PASS
   [ ] If Parallel = ARCANA/AETHER/WITNESS/NEON: Tier must be Rare, Legendary, or Ultimate
       → If Tier = Common AND Parallel ≠ BASE: FAIL — "Parallels reserved for Rare+ only"
   [ ] If Parallel = ONE-OFF: Tier must be Legendary or Ultimate, Council review required
       → Daemon does not auto-generate ONE-OFFs: FAIL — "ONE-OFF requires Council review"

3. ART-DENSITY CLASS ASSIGNED
   [ ] Density class matches tier rubric (§1):
       Common → PLAIN, Rare → MODERATE, Legendary → RICH, Ultimate → MAXIMALIST
   [ ] Prompt includes the density modifier block (§6) for this tier

4. PARALLEL SIGNATURE AUDIT
   [ ] If Parallel = BASE: prompt MUST NOT contain FORBIDDEN PARALLEL SIGNATURES (§2 list)
       → Check prompt for: mandala, sigil, filigree, aurora, nebulae, divine particulate,
         god-rays (non-diegetic), synthwave, neon, cyber, holographic
       → If any forbidden term found: FAIL — "Parallel signature in base art prompt"
   [ ] If Parallel = ARCANA: prompt MUST contain all 4 ARCANA elements (§2)
   [ ] If Parallel = AETHER: prompt MUST contain all 4 AETHER elements (§2)
   [ ] If Parallel = WITNESS: prompt MUST contain all 4 WITNESS elements (§2)
   [ ] If Parallel = NEON: prompt MUST contain all 4 NEON elements (§2)

5. LAMPBLACK CHECK
   [ ] Silhouette layer: the figure's outline is distinctive at 375px width, 2:3 aspect
   [ ] Props layer: at least one anchor object from the universe's iconography list (§2 of
       ART-DIRECTION-V3.md) is explicitly named in the prompt
   [ ] Gesture layer: a narrative beat (moment-of-decision / held-breath / moment-after /
       portent / absent-character-trace) is specified
   [ ] Spine layer: the character's Spine is preserved — if this is a parallel card,
       the Spine event must be legible despite the parallel treatment

6. UNIVERSE VISUAL DNA
   [ ] Prompt names at least one universe-specific visual element (§3 of ART-DIRECTION-V3.md)
       Baker Street: gaslight / fog / Paget / deerstalker / oxblood / Baker Street cobblestone
       Wonderland: Tenniel / blue pinafore / pocket watch / nonsense / dream-logic
       Enchanted Kingdom: Nielsen / Rackham / pre-Disney darkness / forest menace
       Gothic Horror: Coppola / Byzantine / shadow-as-character / ukiyo-e red
       Greek Myth: Waterhouse / Bouguereau / classical drapery / reverent melancholy

7. PROVIDER + TECHNICAL
   [ ] Provider: replicate (NEVER gpt-image-1 for daemon generation)
   [ ] Model: black-forest-labs/flux-1.1-pro-ultra
   [ ] Output: PNG, convert to WebP at quality 85, ≤500KB
   [ ] Aspect: 2:3

8. MANIFEST STATE
   [ ] Entry for this (Moment × Tier × Parallel) exists in commissioning manifest (§7)
       with status "pending"
   [ ] Update manifest entry to "in-progress" before generation begins
   [ ] Update manifest entry to "complete" after successful generation

ALL 8 checks must PASS. Any FAIL blocks generation.
```
