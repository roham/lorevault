# v7.0-marketplace — Iteration 1 Scores

## Changes
- Grid view now uses LARGE cards (260x364 instead of 180x252)
- Card Quick View drawer — click any card to see full detail + price chart + buy without leaving marketplace
- All/Need/Own ownership filter in header — instantly filter to cards you still need
- Sparkline charts in table view and watchlist rows
- Mini card previews in autocomplete suggestions
- Keyboard shortcut: / to focus search, Esc to close
- Owned badge overlay on grid cards
- Removed Link wrapper from grid — opens quick view instead of navigating away (faster workflow)
- Tighter spacing, mono fonts for financial data, denser information hierarchy

## Scores

| Dimension | Score | Notes |
|-----------|-------|-------|
| Search | 7/10 | Text search, faceted filters with counts, autocomplete with card previews, recent searches, saved searches. Missing: fuzzy matching, price range in suggestions |
| Data depth | 7/10 | Price history charts (7d/30d/90d), floor prices, trends, volume, ATH/ATL, sparklines in table. All mock but deterministic. |
| View modes | 7/10 | Grid (large cards), list (dense with sparklines), table (spreadsheet with mini cards + sparklines). Quick view drawer works from all three. |
| Power tools | 7/10 | Watchlist, bulk select, saved searches, Need/Own filter, keyboard /. Missing: price alerts, "buy all watchlist" action. |
| Design | 6/10 | Much improved density. Mono fonts for prices. Quick view is smooth. But recommendation sections still feel casual, not Bloomberg. Need more data density on the main browse view itself. |

**Overall: 6.8/10**
