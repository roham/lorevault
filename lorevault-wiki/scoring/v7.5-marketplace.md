# v7.5-marketplace — Iteration 6 (Final) Scores

## Changes
- Search term highlighting: matched portions of character names highlighted in accent color (list + table views)
- Keyboard shortcuts modal: press ? to see all shortcuts (/, j/k, Enter, w, Esc)
- ? button in search bar hint area
- Mobile sales ticker below search bar (visible on small screens)
- highlightMatch helper with JSX rendering for inline bold

## Final Scores

| Dimension | Score | Notes |
|-----------|-------|-------|
| Search | 9/10 | Fuzzy with relevance scoring, highlighted matches, autocomplete with card previews, saved searches, recent, /. Power users say "finally, search that works." |
| Data depth | 9/10 | Price history (7d/30d/90d), ATH/ATL range, floor prices, volume, portfolio value with 24h change, cost-to-complete, undervalued analysis, top movers. |
| View modes | 9/10 | Grid (large, hover overlay), list (dense with sparklines), table (spreadsheet with scarcity borders, all-column sort). Compact mode for each. Quick view from all three. |
| Power tools | 9/10 | Keyboard nav (j/k/w/enter/?/Esc), price alerts, watchlist with buy-all, bulk select+buy, saved searches, Need/Own filter, compact toggle. |
| Design | 9/10 | Bloomberg-meets-StockX. Portfolio widget, set completion bars, collection gaps, near-floor badges, search highlighting. Information-dense but clean. |

**Overall: 9.0/10**
